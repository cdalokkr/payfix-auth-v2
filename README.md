# PayFix Auth UI v2

Premium, responsive authentication UI for Next.js 16.

## Included
- Login page
- Register page
- Forgot password page
- Reset password page
- Verify email page
- Reusable auth shell
- Base button and input components
- Design system notes

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion-ready structure
- next-themes-ready structure

## Run Locally
```bash
npm install
npm run dev
```

## Pages
- `/`
- `/login`
- `/register`
- `/forgot-password`
- `/reset-password`
- `/verify-email`

## Notes
This starter is built for clean responsive SaaS authentication and can be extended into a full onboarding system.
