# PayFix Auth UI v2 Design System

## Visual Direction
Premium enterprise SaaS authentication UI with soft glass surfaces, dark hero contrast, calm blue primary accents, and responsive layouts across desktop, tablet, and mobile.

## Principles
- Clarity first
- Fast scanning
- Strong hierarchy
- Large touch targets
- Accessible contrast
- Responsive by default
- Motion with restraint

## Core UI Patterns
- Split-screen auth layout on desktop
- Single-column auth layout on tablet and mobile
- Glass card form surfaces
- Hero dashboard preview on the marketing side
- Stepper-based registration flow
- Theme aware components

## Animation Intent
- Subtle page fade-in
- Card lift on hover
- Step transitions with slide and fade
- Soft pulse or float for hero content
- Reduced motion safe fallback

## Responsive Behavior
- Desktop: 2-column split layout
- Tablet: stacked with hero above form
- Mobile: form-first layout with condensed preview content

## Tone
Professional, polished, modern, and enterprise-ready.
